# ATM-interface

A modern and colorful ATM simulation web app built using **HTML, CSS, and JavaScript**.  
It performs basic banking operations like login, deposit, withdraw, transfer, and transaction history tracking.

---

## 🚀 Features

- 🔐 Login System (Demo authentication)
- 💰 Live Balance Display
- ➕ Deposit Money
- ➖ Withdraw Money
- 🔁 Transfer Money
- 📜 Transaction History
- 🚪 Logout Functionality
- 🎨 Clean & Colorful UI Design
- 📱 Responsive Layout

---

## 🛠️ Tech Stack

- HTML5 – Structure  
- CSS3 – Styling  
- JavaScript – Functionality
